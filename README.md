# pwa-client

Progressive Web App frontend for the Lucia privacy-preserving AI agent platform.

## Features

- Built with SvelteKit
- Web Crypto API for E2E encryption (non-extractable session keys)
- E2E encrypted WebSocket communication
- Service worker for push notifications and offline support
- Installable PWA (home screen, standalone window)
- Trust Dashboard for attestation verification

## About Lucia

Lucia is a privacy-preserving AI agent platform that delivers autonomous agent capabilities with hardware-enforced privacy via Trusted Execution Environments (TEE). Learn more at [luciaclaw.com](https://luciaclaw.com).
