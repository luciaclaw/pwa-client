# Contributing to pwa-client

Thank you for your interest in contributing to Lucia!

## Getting Started

1. Fork the repository
2. Create a feature branch (`git checkout -b feat/my-feature`)
3. Make your changes
4. Run tests (`npm test`)
5. Commit using [conventional commits](https://www.conventionalcommits.org/) (e.g., `feat:`, `fix:`, `docs:`)
6. Open a pull request

## Code of Conduct

Be respectful and constructive. We're building privacy-preserving technology for everyone.
